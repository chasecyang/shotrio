# Project-local Codex skills

This folder contains Codex skills migrated for this project (some originated from `.claude/commands` and `.claude/agents`).

## Enable (pick one)

1. Project-local (recommended)
   - From repo root: `export CODEX_HOME="$PWD/.codex"`
   - Restart Codex
2. Global install
   - Copy to: `~/.codex/skills/`
   - Restart Codex

## Usage

- Commits: `$git-commit-helper`
- Refactoring: `$code-simplifier`
- UI design: `$frontend-design`
