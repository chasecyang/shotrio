# Beta 标识视觉效果说明

## 组件样式预览

### 1. BetaBadge（徽章）

#### Default 变体（醒目）
```
┌─────────┐
│ • BETA  │  ← 主色调边框和背景
└─────────┘     橙色系，较醒目
```
- 边框：`border-primary/30`
- 背景：`bg-primary/10`  
- 文字：`text-primary`
- 使用场景：登录页、需要引起注意的位置

#### Subtle 变体（柔和）
```
┌─────────┐
│ · BETA  │  ← 灰色边框和背景
└─────────┘     低调柔和
```
- 边框：`border-muted-foreground/20`
- 背景：`bg-muted/30`
- 文字：`text-muted-foreground`
- 使用场景：全站 Header Logo 旁

#### Minimal 变体（极简）
```
  BETA      ← 无边框无背景
```
- 边框：无
- 背景：透明
- 文字：`text-muted-foreground/60`
- 使用场景：编辑器 Header，不干扰操作

## 颜色方案（OKLCH）

### 浅色模式
- Primary: `oklch(0.55 0.18 40)` - Burnt Sienna（深橙色）
- Primary/10: 10% 透明度 - 徽章背景
- Primary/30: 30% 透明度 - 徽章边框
- Muted: `oklch(0.94 0.02 85)` - 暖灰色

### 深色模式
- Primary: `oklch(0.68 0.20 40)` - Sunset Orange（亮橙色）
- 背景更深，但保持温暖的视觉感受

## 字体设计

### BetaBadge 文字
- 字体：`font-mono`（等宽字体）
- 大小：`text-[10px]`（超小号）
- 字重：`font-semibold`（半粗）
- 字距：`tracking-wider`（宽字距）
- 转换：`uppercase`（大写）

**效果**：`B E T A` - 技术感强，清晰易读

### BetaBanner 文字
- 字体：系统默认 sans-serif
- 大小：`text-sm`（小号）
- 强调文字：`font-semibold text-primary`
- 说明文字：`text-muted-foreground`

## 动画效果

### BetaBanner 进入动画
```
开始: translateY(-100%)
结束: translateY(0)
时长: 300ms
缓动: ease-out
```

使用 Tailwind 类：`animate-in slide-in-from-top duration-300`

### 关闭动画
```
开始: opacity(1)
结束: opacity(0) + display(none)
时长: 200ms
```

## 响应式适配

### 桌面端（≥768px）
- 徽章：完整显示
- 横幅：完整文案 + 图标 + 关闭按钮
- Header：Logo + 徽章并排

### 移动端（<768px）
- 徽章：保持显示但可能缩小
- 横幅：文案可能换行，保持可读性
- Header：Logo 和徽章垂直对齐或缩小

## 可访问性

### 语义化
- 使用 `<Badge>` 组件（基于 `<span>`）
- 关闭按钮包含 `aria-label="Dismiss beta notice"`

### 颜色对比度
- Default 变体：≥ 4.5:1（符合 WCAG AA）
- Subtle 变体：≥ 3:1（符合 WCAG A）
- Minimal 变体：≥ 3:1（低优先级内容）

### 键盘导航
- 关闭按钮可通过 Tab 聚焦
- Enter/Space 可触发关闭

## 性能考虑

### LocalStorage
- 读取：仅在客户端组件 mount 时
- 写入：仅在用户点击关闭时
- Key 命名：前缀统一，便于清理

### 渲染优化
- BetaBanner 使用状态控制显示/隐藏
- 关闭后不渲染 DOM（return null）
- 避免不必要的重渲染

## 浏览器兼容性

### 全面支持
- ✅ Chrome/Edge 90+
- ✅ Firefox 88+
- ✅ Safari 14+
- ✅ 所有现代移动浏览器

### 优雅降级
- 如果 localStorage 不可用：横幅始终显示
- 如果不支持 OKLCH：回退到 RGB 颜色
- 如果不支持动画：直接显示，无动画
